## Что нужно для работы

1. Установить Python 3.13
2. Установить iVCam на телефон и компьютер
3. Подключить телефон к ПК через USB и запустить iVCam
4. Установить зависимости: `pip install -r requirements.txt`
5. Запустить: `python main.py`

## Как установить iVCam

- Скачать на ПК: https://www.ivcam.cc
- Скачать на телефон: Google Play или App Store
- Подключить телефон через USB
- На телефоне включить USB-отладку (режим разработчика)
- Запустить iVCam на ПК и телефоне
